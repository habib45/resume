Banglalink ( MY BL-App )

I was involved in a project with Banglalink, the second largest telecom industry in Bangladesh, working on the development of their MY BL-App. This app has daily active users of around 500,000 and a total Active user base exceeding 5 million. It offers over 100 functionalities, including recharge options and various services. |

Technical Responsibility:

  • Design and Build application from scratch
  • Establish testing platform, mainly focused on providing maximum feasible test coverage for business logics.
  • Perform complex diagnosis and come up with optimized solution since this app has a large user base.
  • Use Firebase Analytics to monitor and track user activity.
  • Collaborate with client to lock features for release, share case studies and impact on business.Banglalink is the 2 nd largest telecom industry in Bangladesh having around daily 500k active app users with above 5 million total. 100+ functionalities are available through this app with recharge
  • Tools and Technology: PHP Laravel ( Backend & CMS ), React js ( website),iOS, Swift, iOS Architecture components, MVVM Design Pattern, Core Data, RX Swift, Google Map, Widgets, Firebase, Unit Testing.

Project Technical Details:
